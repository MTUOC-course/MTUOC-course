# Overview

Machine translation technologies are useful when they can be deployed, connected to real translation workflows and evaluated in a reliable way. This module introduces these three practical dimensions through the MTUOC ecosystem.

You will start by looking at different ways of deploying machine translation systems. You will then learn how to start translation models with MTUOC Server, including both neural machine translation (NMT) models and LLM-based translation models. The module then moves from deployment to use: you will explore how MT systems can be integrated into translation workflows and how MTUOC Translator can be used as an interface for working with machine translation.

The final part of the module focuses on automatic evaluation. You will learn why evaluation is necessary, what automatic metrics can and cannot tell us, and how tools such as SacreBLEU and COMET can support the comparison of machine translation outputs. Automatic evaluation is presented as a practical method for obtaining reproducible, scalable results, while keeping in mind that human evaluation remains essential for assessing aspects such as nuance, adequacy and style.

By the end of this module, you will be able to:

- understand different deployment options for machine translation systems;
- start NMT models with MTUOC Server;
- start LLM-based translation models for translation tasks;
- connect machine translation systems to translation workflows;
- use MTUOC Translator as a translation interface;
- evaluate machine translation output with automatic metrics;
- interpret automatic evaluation results critically.

## Module Roadmap

This module is organised into the following sections:

1. **Deployment**  
   Introduction to the main ways in which machine translation systems can be deployed and accessed.

2. **Starting NMT Models with MTUOC Server**  
   Practical guidance on starting neural machine translation models using MTUOC Server.

3. **Starting LLM-Based Translation Models**  
   Introduction to using LLM-based systems for translation through MTUOC tools.

4. **Integration into Translation Workflows**  
   Explanation of how machine translation systems can be connected to professional translation environments and workflows.

5. **MTUOC Translator**  
   Practical use of MTUOC Translator as an interface for working with machine translation systems.

6. **Automatic Evaluation**  
   Introduction to automatic MT evaluation and the use of metrics such as SacreBLEU and COMET.

7. **Practical Activity**  
   A hands-on task to apply the main concepts and tools covered in the module.

8. **Self-Assessment**  
   A short quiz to review the key concepts before moving on to the next module.

We recommend completing the sections in the order presented.
