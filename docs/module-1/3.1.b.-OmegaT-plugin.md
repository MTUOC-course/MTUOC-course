## 1. Installation

* Download the plugin: [MtuocTranslatePlugin-0.1.0.jar](https://github.com/aoliverg/MTUOC-OmegaT-plugin/releases/download/v0.1.0/MtuocTranslatePlugin-0.1.0.jar). 
* Place the JAR file in the corresponding directory for your operating system:

  * Windows: `C:\Program Files\OmegaT\plugin`
  * Linux: `$HOME/.omegat/plugin`
  * Mac: `~/Library/Preferences/OmegaT/plugins`

**Note**: These paths may vary depending on your specific OmegaT installation and configuration.

## 2. Plugin configuration

Once the JAR file is in the correct folder, launch OmegaT and follow these steps:

1. Navigate to **Options > Preferences**.

<img width="590" height="370" alt="image" src="https://github.com/user-attachments/assets/f2c47d8f-93dd-40a9-a6fc-8bff123c3c31" />

2. In the sidebar, select **Machine Translation**.

3. Select **MTUOC MT Engine** and click **Configure**. 

<img width="277" height="170" alt="image" src="https://github.com/user-attachments/assets/d14783a0-32a1-459f-8950-f9696c23001c" />

4. Enter the details for your available system (either a local or remote server).

5. Click **Test connection** to verify the setup. If the configuration is correct, the message "Server connection OK" will be displayed.

The MT engine is now active, and suggestions will appear automatically in the Machine Translation pane.

For further details on integration settings, please consult the OmegaT manual: [Machine translations](https://omegat.sourceforge.io/manual-standard/en/chapter.panes.html#panes.machinetranslation)
