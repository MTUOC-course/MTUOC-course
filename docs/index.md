# MTUOC: A Practical Course on Machine Translation Technologies

Welcome to this open online course on practical machine translation technologies that introduces learners to the use of **MTUOC**, a modular ecosystem of open-source tools designed to make machine translation workflows more accessible to translators, language professionals, teachers, researchers and students.
The course is organised into four modules:

1. **Deployment, Integration and Evaluation**
2. **Corpus Engineering**
3. **NMT Training and Fine-tuning**
4. **LLMs for Translation**
