# How to Navigate This Course

This page explains how the course is organised and how to make the most of the learning experience.

## Before You Begin

To complete the practical activities in this course, you should:

- have basic computer literacy;
- be comfortable downloading and installing software;
- be willing to follow step-by-step technical tutorials.

No previous programming experience is required.

Some practical activities require Docker Desktop and other software components. Installation instructions are provided when they are first needed in the course.

---

## Course Structure

The course is organised into four thematic modules. Each module follows a similar structure to help you progress through the course.

A typical module includes:

- an introduction to the topic;
- learning objectives;
- learning materials;
- practical tutorials;
- hands-on activities;
- a self-assessment quiz;
- additional resources, where appropriate.

The course is self-paced, but we recommend completing the modules in order, as each module builds on concepts introduced previously.

## Recommended Learning Path

For each module, we recommend following this sequence:

1. Read the introduction and learning objectives.
2. Study the learning materials.
3. Complete the practical tutorials.
4. Carry out the proposed activities.
5. Take the self-assessment quiz.
6. Join the discussion on Discord.
7. Continue to the next module.

---

## Self-Assessment

Each module concludes with a short self-assessment quiz designed to help you review the main concepts before continuing.

The quizzes are intended to support your learning. You may repeat them as many times as you wish.
