    3.2.a. MTUOC‐Trados‐plugin

    Downloading the plug-in

    Download the plug-in for your version of Trados Studio. At the moment we offer a plug-in for each of the following versions: 2019, 2021 and 2022. Once downloaded, unzip the file.

    Installing the plug-in

    Uninstall any previous versions of the MTUOC Trados plugin before installing the new version. Open the uncompressed folder and doble-clic to the file:

    The following interface will appear:

    Choose the correct version of Trados Studio and the folder (for all the domain computers, current user or all users) where the plug-in will be installed. Clic on the Next button. Once the installation is completed click on the Finnish button.

    Unsigned plug-in warning

    Depending on the version of the plug-in, when opening Trados Studio, a warning indicating that the plug-in is unsigned may appear. As the following one:

    Answer Yes.

    The plug-ins are signed by Trados and we ask for the signature once a new version of the plug-in is released. As this process takes some days, you may eventually find unsigned plug-ins.

    Plug-in configuration

    To configure the plug-in, once opened a project go to Project Configuration and select Translation memory and machine translation, either for All language combination or for a given languae combination. Then select Use and clic on MTUOC-MACHINE TRANSLATION and this plug-in will appear:

    Unselect it, in case it's selected, and then clic on Settings. The following interface will appear:

    General tab

    In this tab we can set the IP (or server name) and Port of the MTUOC server. No language pair is provided as each IP and port serves an unique language pair. The plug-in, nor Trados, make no language verification, so the user should know the language pair of the IP and port combination.

    Connection test tab

    In this tab we can perform a connection test to make sure the server works properly. Write or paste a source sentence and clic on the button Send

    Advanced tab

    In this tab some advanced configuration can be set:

    • Cache - Keep the cache (days): to set how many days the cache sould be kept. You can also delete the cache at any moment clicking on the Delete cache button.
    • Translation configuration:
    • Score: the score given to the machine translation. This allows to combine machine translation with translation memories. The machine translation target will be selected of the best translation memory match is under this score.
    • Description: the description string that will be set in the SDLXLIFF file of the project for the translated segments coming from machine translation. The following values can be set: Unknown, TM, Machine translation, Alignment, Context TM, Adaptative Machine Translation, Nmt and AutomaticTranslation.

    To accept the configuration the button Accept can be clicked in any tab.

    To close and accept the project settings click on the button OK.

    Once the configuration is done, the source segments will be translated with the MTUOC server set in the configuration.

    Troubleshooting

    Uninstall the plugin

    Using the Plugin Installer

    Your Trados Studio distribution comes with a program that facilitates the installation and uninstallation of plugins. The program is called Sdl.Community.SdlPluginInstaller.exe. Depending on the version this program can be located in different folders on your computer.

    For example, in Trados Studio 2022 the program most probably is located at: Program Files (x86)> Trados > Plugin installer. Once located, double-click on it and the following (or similar) interface would appear:

    In this interface, be sure the correct version of Trados is selected in Filter by Studio version:". The plugin will appear and you can uninstall it clicking on the Uninstall** button at the right of the plugin.

    For other versions of Trados Studio, for example the 2021, the program can be located at: Program Files (x86)> SDL > Plugin installer.

    Manually uninstalling the plugin

    You can also manually uninstall the plugin deleting some files and folders in your computer. The location of these files and foldes may depend on the version and the installation.

    For Trados Studio 2022, try locating the following folder. Under the User's folder, locate the AppData folder and go to \AppData\Roaming\Trados\Trados Studio\17\Plugins. Under this folder, you'll find two subfolders: Packages and Unpacked. In the Unpacked folder, delete the SDL.Trados.MTUOC folder. If this plugin is the only one installed in the system, you can safely delete all the Unpacked folder. Then, go to the Packages folder and delete all the files and folders corresponding to this plugin.

    For Trados Studio 2021, follow the same procedure in the folder \AppData\Roaming\SDL\SDL Trados Studio\16\Plugins.

    For Trados Studio 2019, follow the same procedure in the folder \AppData\Roaming\SDL\SDL Trados Studio\15\Plugins.

    Manual installation of the plugin

    The plugin can manually installed following these steps:

    1. Download the zip file corresponding to your version of Trados Studio.

    2. Uncompress the zip file.

    3. Copy all the files and folders to the following folders, depending on the version of Trados Studio:
    4. For Trados Studio 2019, on your user's folder in \AppData\Roaming\SDL\SDL Trados Studio\15\Plugins\Packages
    5. For Trados Studio 2021, on your user's folder in \AppData\Roaming\SDL\SDL Trados Studio\16\Plugins\Packages
    6. For Trados Studio 2022, on your user's folder in \AppData\Roaming\Trados\Trados Studio\17\Plugins\Packages
